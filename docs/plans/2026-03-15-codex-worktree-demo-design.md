# Codex Worktree Demo Design

**Goal:** Write a beginner-friendly Markdown tutorial that demonstrates how to use Git worktrees with Codex Desktop.

**Audience:** New users who want to get started quickly and need a guided walkthrough more than a deep conceptual explanation.

**Chosen approach:** Tutorial-first, with a small amount of explanation mixed into the flow. The document should explain what a worktree is, then walk through a full example from creating a worktree to cleaning it up.

**Outline:**
- Brief introduction to what a worktree solves
- Prerequisites
- Step-by-step walkthrough
- Short explanation of why this is useful in Codex Desktop
- Common questions and pitfalls

**Notes:**
- Keep the wording simple and practical
- Prefer copyable commands
- Focus on the user action path, not Git internals
